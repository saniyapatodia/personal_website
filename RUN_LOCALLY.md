# How to Run Your Website Locally

## Quick Start (Easiest)

1. **Open Terminal** (Press `Cmd + Space`, type "Terminal", press Enter)

2. **Navigate to your website folder:**
   ```bash
   cd /Users/saniyapatodia/Desktop/Docs/Extra-Curriculars/personal_website
   ```

3. **Start a local server:**
   ```bash
   python3 -m http.server 8000
   ```

4. **Open your browser** and go to:
   ```
   http://localhost:8000
   ```

5. **To stop the server:** Press `Ctrl + C` in the terminal

---

## Alternative: Open Directly (May have limitations)

You can also just double-click `index.html` to open it in your browser, but some features (like images) might not work perfectly due to browser security restrictions.
