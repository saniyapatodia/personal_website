# personal website

Static site — landing page, about, experience, and what I’m building. Hosted on GitHub Pages.

There’s a little photo thing on desktop if you move the mouse on the landing screen (mostly for fun).

## What’s in here

- `index.html` — copy and structure
- `styles.css` — layout and colours
- `script.js` — section switching and the hover photos
- `images/` — JPEGs used for those photos (`script.js` lists the filenames)

## Run it locally

Open `index.html` in a browser, or use any static server from this folder.

## Photos

Drop images in `images/` and add the paths to the `photoPaths` array in `script.js`. See `images/README.md` for a bit more detail.
